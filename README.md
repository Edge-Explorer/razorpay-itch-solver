## 📖 What Is This Repository?

This is **not** a tutorial project. This is **not** a generic portfolio filler.

This repository is a growing collection of **AI-powered solutions** built specifically for problems validated by real Indians on [Razorpay's Fix My Itch](https://razorpay.com/m/fix-my-itch/) — a platform that has gathered **10,000+ verified real-world pain points** reported by **50,000+ people** across India.

Each sub-folder in this repo is an **independent, deployable project** targeting one specific "itch." Every solution is built using modern AI — RAG pipelines, LLMs, and autonomous AI agents.

## 🧠 How We Work (The Process)

For each problem, we follow a structured brainstorming and development workflow:

| Phase | What Happens |
|-------|-------------|
| **1️⃣ Problem Discovery** | Select a problem from Fix My Itch with a meaningful Itch Score |
| **2️⃣ Brainstorming** | Deep dive into: *What exactly is the problem? Who faces it? What do they wish existed?* |
| **3️⃣ Architecture Design** | Define the AI approach (RAG / Agent / LLM pipeline), draw system diagrams |
| **4️⃣ Implementation** | Build the solution with clean, documented code |
| **5️⃣ Documentation** | Write a problem-specific README that tells the full story |

## 🛠️ AI Tech Stack

Every project uses a combination of the following, chosen based on what fits the problem best:

- **LLMs**: Google Gemini, OpenAI GPT, Anthropic Claude
- **RAG**: LangChain, ChromaDB, FAISS, Pinecone
- **Agents**: LangGraph, CrewAI
- **Backend**: Python, FastAPI
- **Frontend**: Streamlit (rapid demos), Next.js (polished products)
- **Diagrams**: Mermaid.js for all system architecture docs
